# Java Calculator App

![Java](https://img.shields.io/badge/Java-ED8B00?style=for-the-badge&logo=openjdk&logoColor=white)
![Gradle](https://img.shields.io/badge/Gradle-02303A?style=for-the-badge&logo=gradle&logoColor=white)

A simple calculator application built using **Java** and **Gradle**.  
This project demonstrates basic arithmetic operations and the use of **Gradle as a build automation tool**.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Command line based interaction
- Gradle build automation

---

## 🛠 Tech Stack

- **Java**
- **Gradle**
- **CLI (Command Line Interface)**

## ▶️ How to Run

### 1️⃣ Clone the repository

```
git clone https://github.com/anshdev02/Calculator-App.git
```

### 2️⃣ Navigate to the project folder

```
cd Calculator-App
```

### 3️⃣ Build the project

```
gradle build
```

Gradle will compile the Java code and create build files.

---

### 4️⃣ Run the application

```
gradle run
```

This will execute the main Java class and start the calculator program.
