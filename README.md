# Java Calculator App

A simple calculator application built using **Java** and **Gradle**.

This project demonstrates basic arithmetic operations and the use of a Gradle build system for managing dependencies and compiling the project.

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Gradle build automation

---

## 🛠 Tech Stack

- Java
- Gradle
